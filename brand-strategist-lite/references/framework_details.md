# Brand Strategist Lite - Framework Details

## 1. Jobs to be Done (JTBD) Framework
JTBD focuses on the "job" a customer hires a product to do.

### Core Components:
- **Functional Job**: What practical task does the user want to accomplish? (e.g., "Mow the lawn")
- **Emotional Job**: How does the user want to feel? (e.g., "Feel proud of my home")
- **Social Job**: How does the user want to be perceived? (e.g., "Seen as a responsible neighbor")

### Analysis Question:
- "When [situation], I want to [motivation], so I can [expected outcome]."

## 3. Brand Archetypes (The 12 Archetypes)
Brand personalities that connect with human desires.

1.  **The Innocent**: Strives for happiness, purity, and simplicity.
2.  **The Everyman**: Connects with others, down-to-earth, supportive.
3.  **The Hero**: Proves worth through courageous acts, improves the world.
4.  **The Outlaw**: Disrupts status quo, shock value, liberation.
5.  **The Explorer**: Seeks new experiences, autonomy, freedom.
6.  **The Creator**: Creates something new, vision, expression.
7.  **The Ruler**: Control, creates a prosperous, successful family or community.
8.  **The Magician**: Makes dreams come true, transformation.
9.  **The Lover**: Intimacy, experience, passion.
10. **The Caregiver**: Protects and cares for others, service.
11. **The Jester**: Live in the moment with full enjoyment, fun.
12. **The Sage**: Seeks truth, knowledge, wisdom.

## 4. Master Report Templates (3C Framework)

### User Insights Report (Customer 面向)
**1. Jobs to be Done (JTBD) Analysis**
-   **Functional Jobs**: Practical tasks the user wants to solve.
-   **Emotional Jobs**: Inner feelings and psychological states desired.
-   **Social Jobs**: External perception and identity management.
-   **JTBD Statement**: "When I [context], I want to [motivation], so I can [outcome]."

**2. Bain 30 Value Elements Mapping (Demand - $W_u$)**
-   **Mapping Table**: Category (Functional/Emotional/etc.), Element, User Evidence, **User Weight ($W_u$) 1-10**.

**3. Customer Psychographics & Persona**
-   **Who they are**: Detailed profile including values, lifestyle, and professional context.
-   **Expected Archetype/Personality**: The specific brand personality the user group anticipates or trusts (e.g., The Sage, The Hero).

---

### Market Differentiation Report (Competitor 面向)
**1. Individual Competitor Analysis (Up to 3)**
-   **Functional Specs**: Listing core features for each competitor.
-   **Functional Differentiation**: Comparative analysis of product strengths.

**2. Bain 30 Value Elements Mapping (Supply - $S_c$)**
-   **Mapping Table**: Category, Element, **Comp A Score**, **Comp B Score**, **Comp C Score** (1-10).

**3. Multi-Archetype Mapping**
-   **Archetype Identify**: Primary and Secondary status for each competitor.
-   **Evidence**: Style/Copywriting evidence for each brand.

---

### Final Brand Strategy (Corporation 面向)
**1. Brand Identity & Product Strength**
-   **Core Mission**: The brand's foundational "Why."
-   **Functional Moat**: Specific product features or processes that provide a unique competitive advantage.

**2. Expanded 3C Competitive Matrix (Quantitative)**
-   **Comparison Table**: 
    | Category | Element | $W_u$ | Comp A | Comp B | Comp C | **Our Score ($S_{corp}$)** | **Max Gap** | **$O$ Score** |
    | :--- | :--- | :---: | :---: | :---: | :---: | :---: | :---: | :---: |
-   *Max Gap*: Calculated as $S_{corp} - \text{Max}(S_{comp\_A}, S_{comp\_B}, S_{comp\_C})$.
-   *Goal*: To prove superiority against the toughest rival in each high-value zone.

**3. 3C Sweet Spot Analysis**
-   **Customer Want**: Top unmet needs (High $W_u$, Low Market $S_c$).
-   **Competitor Own**: Which competitor dominates which element.
-   **Our Strategic Pivot**: Where our high $S_{corp}$ beats the entire field.

**4. Archetype & Personality (Derived from 3C)**
-   **Target Archetype**: Derived from the Sweet Spot.
-   **Communication Strategy**: Tone, Pillars, Taglines.

**5. Implementation Roadmap**
-   Sequential action items.

## 5. Quantitative Analysis Logic
We use a weighted 3C scoring system to identify market winners.

### A. User Weight ($W_u$)
How important is this element to the user? (1-10)

### B. Competitor & Corporation Scores ($S_c$, $S_{corp}$)
How well is the element delivered? (1-10)

### C. Strategic Indicators
1.  **Opportunity Score ($O$)**: $$O = (W_u \times 1.5) - \text{Avg}(S_c)$$
    -   *Identifies general market underservice.*
2.  **Competitive Gap (Gap)**: $$\text{Gap} = S_{corp} - \text{Max}(S_c)$$
    -   *Identified our absolute superiority over all analyzed rivals.*

### D. Symbolic Matrix
Used for quick visual comparison.
- `●`: Strong/Core (Score 8-10)
- `◐`: Present (Score 4-7)
- `○`: Weak/Absent (Score 1-3)
- `🚀`: **Win Zone** (High $W_u$ + Positive Gap)
